# hy-ts-react-music

一个基于React + TypeScript的音乐播放器项目，具有现代化的UI设计和完整的音乐播放功能。

## 项目亮点

### 技术栈

- **React 19.1.1**：使用最新版本的React框架
- **TypeScript 4.9.5**：提供完整的类型支持，增强代码可维护性
- **Redux Toolkit 2.8.2**：现代化的状态管理方案
- **React Router 7.7.1**：实现页面路由和导航
- **Ant Design 5.27.0**：提供丰富的UI组件
- **Styled Components 6.1.19**：组件样式解决方案
- **Axios 1.11.0**：处理API请求
- **ESLint + Prettier**：确保代码质量和一致性

### 项目结构

- **组件化设计**：将UI拆分为多个可复用组件
- **模块化架构**：清晰的目录结构，便于维护
- **懒加载路由**：优化首屏加载速度
- **状态管理**：使用Redux Toolkit管理全局状态
- **服务层**：统一处理API请求
- **工具函数**：提供通用功能支持
- **主题配置**：支持全局主题管理

### 功能模块

- **发现音乐**：包含推荐、专辑、歌手、排行榜、电台、歌曲等子模块
- **我的音乐**：个人音乐管理
- **关注**：关注内容管理
- **下载**：音乐下载功能
- **音乐播放器**：支持播放控制、进度调节、歌词显示等

### 性能优化

- **代码分割**：使用React.lazy实现组件懒加载
- **路由懒加载**：按需加载路由组件
- **状态管理优化**：使用Redux Toolkit的createSlice减少模板代码

### 开发体验

- **热更新**：开发过程中实时预览变化
- **代码规范**：使用ESLint和Prettier确保代码质量
- **类型安全**：TypeScript提供类型检查
- **构建优化**：使用craco配置构建过程

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm start
```

### 构建生产版本

```bash
npm run build
```

### 运行测试

```bash
npm test
```

## 项目结构

```
src/
├── assets/           # 静态资源
│   ├── css/          # 样式文件
│   ├── data/         # 本地数据
│   ├── img/          # 图片资源
│   └── theme/        # 主题配置
├── components/       # 通用组件
├── router/           # 路由配置
├── service/          # API服务
├── store/            # 状态管理
├── utils/            # 工具函数
├── views/            # 页面组件
├── App.tsx           # 应用入口组件
└── index.tsx         # 应用挂载点
```
